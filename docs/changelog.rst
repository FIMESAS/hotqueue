Changes in v0.2.0
=================
- Renamed ``dequeue()`` method on ``HotQueue`` to ``get()``
- Renamed ``enqueue()`` method on ``HotQueue`` to ``put()``
- Added ``HotQueue.worker()`` function decorator
- ``HotQueue.get()`` method now supports block and timeout arguments
- Added test suite
